✨ Powerful Features

🧠 Conversational AI

Natural language understanding and generation using transformer models for meaningful conversations.

🎭 Personality Traits

Adjustable personality (helpfulness, creativity, empathy, humor, formality) for personalized interactions.

🚀 Quick Start

Clone the Repository
git clone https://github.com/jayravey3-lang/Dagon-.git
cd Dagon-
Create Virtual Environment
python -m venv venv
source venv/bin/activate # On Windows: venv\Scripts\activate
Install Dependencies
pip install -r requirements.txt
Run Dagon AI
# Basic Chat
python examples/basic_chat.py

# Enhanced Chat
python examples/enhanced_chat.py

# API Server
python -m dagon.server

Basic Usage Example

from dagon import DagonAI

# Initialize Dagon
dagon = DagonAI()

# Have a conversation
response = dagon.chat("Hello! What can you do?")
print(response)

Advanced Usage

from dagon.enhanced import EnhancedDagonAI

# Initialize enhanced Dagon
dagon = EnhancedDagonAI()

# Configure personality
dagon.set_personality_trait("empathy", 0.95)

# Add knowledge
dagon.add_knowledge("greeting", "response", "Hello!")

# Get statistics
stats = dagon.get_statistics()

🔌 REST API Endpoints

Start the API server with: python -m dagon.server

Endpoint Method Description
/health GET Health check endpoint
/chat POST Send a message and get a response
/personality GET Get current personality profile
/personality PUT Update personality trait
/statistics GET Get conversation statistics
/knowledge POST Add knowledge to knowledge base
/memory GET Recall stored memories
/reset POST Reset conversation

Example API Request

curl -X POST http://localhost:5000/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Hello Dagon!"}'
